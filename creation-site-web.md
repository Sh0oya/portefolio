# Création de site web : un site qui ramène des clients, pas une vitrine de plus.

> **Mathieu Haye crée des sites web sur mesure pour les PME et les indépendants : sites vitrines, landings de conversion et refontes, pensés pour être trouvés sur Google et cités par les IA (ChatGPT, Perplexity, Google AI) dès le premier jour.** Si un site simple sur WordPress ou Webflow vous suffit, il vous le dira. Le sur-mesure doit se justifier.

Trois façons de démarrer :

- **Site vitrine efficace** : les pages essentielles, réservation ou contact intégré, en ligne rapidement. Version simple dès 400 €, livrée en une semaine, périmètre cadré.
- **Landing de conversion** : une page, un objectif, des visites et des conversions mesurées. Dès 400 €, livrée en une semaine.
- **Refonte** : moderniser le design et les contenus sans perdre le référencement existant. Sur devis, après un cadrage de 30 minutes.

La méthode tient en trois étapes : cadrage en 30 minutes, maquette validée ensemble, mise en ligne avec suivi. Le référencement est prévu dès la construction : structure claire, données structurées, fichiers lisibles par les IA. La preuve : mathieuhaye.fr est construit ainsi et sert de chantier vitrine. Testez gratuitement votre site actuel sur [Visible par les IA](https://mathieuhaye.fr/visible-par-les-ia).

Le cadre : le code vous appartient, RGPD respecté, hébergement en France, un seul interlocuteur du cadrage à la maintenance.
